# Simple CLI File Manager App Create using Python.

A lightweight, Python-based Command Line Interface (CLI) application for basic file operations. This project demonstrates CRUD (Create, Read, Update, Delete) functionality using Python's native `os` and `io` modules.

##  Features

- **Create:** Initialize new files safely using exclusive creation mode (`x`).
- **Read:** Display file contents directly in the terminal.
- **Append:** Add new data to existing files without overwriting previous content.
- **Delete:** Remove unwanted files with built-in error handling.
- **List:** View all available files in the current working directory.

##  Tech Stack

- **Language:** Python 3.x
- **Modules:** `os` (Operating System interface)
- **Environment:** Developed and tested on Linux (Fedora)

# Run

   python main.py
